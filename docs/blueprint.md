# **App Name**: DayView

## Core Features:

- Clock Display: Display the schedule as two circular clock diagrams, one for day time (6 AM - 6 PM) and one for night time (6 PM - 6 AM).
- Markdown Input: Allow users to input and edit tasks and categories in a Markdown format. The tasks must include start time, end time, category, and status
- Task Checklist: Display a checklist of tasks alongside the clock diagrams.

## Style Guidelines:

- Color palette should follow Material Design principles.
- Use distinct colors for each task category for easy visual differentiation.
- Accent color: Teal (#008080) to highlight interactive elements.
- Clock diagrams should occupy the top portion of the screen with the task checklist below.
- Ensure a responsive layout for different screen sizes.
- Use simple, clear icons for task categories.
